# フリーランス会計アプリ（学習用）概要

## 目的
Django 学習を目的とした、フリーランスの売上・経費を管理する会計アプリ。
自身のフリーランス経験を題材にすることで、データのイメージが湧きやすくモチベーションを維持しやすい。

## 位置づけ
- 家計簿アプリ（Category + Transaction の2テーブル構成）の発展版
- 「取引先（案件）」という3つ目の軸が加わり、複数の外部キー（リレーション）を扱う練習になる
- 骨格は家計簿版と同じ。これまでの手順（Phase 0〜6）はそのまま流用可能

## 学習ポイント
- 必須の外部キー（Category）と任意の外部キー（Client, `null=True, blank=True`）の違い
- ORM の集計（月別集計、カテゴリ別集計、取引先別集計）
- 集計結果を Python 側で加工してグラフ用データを組み立てる（利益 = 売上 − 経費）

## 注意事項
- 本アプリの科目分類は確定申告の勘定科目に「寄せる」だけであり、実際の申告区分そのものではない
- 本リポジトリは **学習用の public リポジトリ**。機密情報・実データは絶対にコミットしない（→ `05_security.md`）
